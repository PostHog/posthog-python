# Semantic Kernel + PostHog AI Examples

Track Semantic Kernel LLM calls with PostHog.

## Setup

```bash
cp .env.example .env
# Fill in your API keys in .env
uv sync
```

## Examples

- **chat.py** - Semantic Kernel with PostHog tracking via AsyncOpenAI wrapper

## Run

```bash
source .env
uv run python chat.py
```
